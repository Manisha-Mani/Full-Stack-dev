Jquery Query

JQuery is a concise JS library which is widely used. It's motto is "Write less, do more".

Features

  • Easy to manipulate DOM
  • Wide community
  • Supports various plugins-like AJAX
  • Light weight-19KB after compression
  • Cross browser suppot
  • Simplifies event handling
  • Has more animations which helps in creating interactive webpages
Installation
  • local installation-copy entire code from Jquery website and paste in in .js file on your folder. Not good practice as we may tend to change something accidently.
  • link to CDN(Content Delivery Network): you can add jQuery library in your HTML code from CDN

DOM

Document Object model is the tree structure of HTML elements.

JQuery Selectors

They are used to select elements of DOM. DOM should be easily manipative, thus selectors are used. Syntax-$("element").method

JQuery methods

They are used to manipulate selected elements

  • before-$("element").before("content")-applies content before selected element
  • aftee-$("element").after("content")-applies content after selected element
  • text-return/set content:$("element").text()-returns the content of element, $("element").text("content")-sets the content to the element
  • html-similar to text method except that it returns the html content with element content(like li tag)
  • css-$("element").css("property","value")-styles the element. sets or returns the style property
